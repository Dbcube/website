A realistic, end-to-end slice of a blogging backend. Everything here is the kind
of code you'd actually ship: typed rows, a publish workflow, eager-loaded
relations and pagination — written in TypeScript.

## The types

In a real project these come from `npx dbcube generate` (it reads your `.cube`
schema and writes `dbcube/types.ts`). Shown inline so the example is self-contained.

```ts
// dbcube/types.ts  (generated)
export interface Author {
  id: number;
  name: string;
  email: string;
  bio: string | null;
}

export interface Post {
  id: number;
  author_id: number;
  title: string;
  slug: string;
  body: string;
  status: "draft" | "published";
  views: number;
  published_at: string | null;
  created_at: string;
}

export interface Comment {
  id: number;
  post_id: number;
  author_name: string;
  body: string;
  created_at: string;
}
```

## Creating a post (draft by default)

```ts
import { dbcube } from "dbcube";
import type { Post } from "./dbcube/types";

const db = dbcube.database("blog");

function slugify(title: string): string {
  return title.toLowerCase().trim().replace(/[^a-z0-9]+/g, "-").replace(/^-|-$/g, "");
}

export async function createDraft(authorId: number, title: string, body: string): Promise<Post> {
  const [post] = await db.table<Post>("posts").insert([
    {
      author_id: authorId,
      title,
      slug: slugify(title),
      body,
      status: "draft",
    },
  ]);
  return post; // insert returns the row with its generated id
}
```

## Publishing a post

A scoped update — Dbcube refuses updates without a `where()`, so you can't
accidentally publish the whole table.

```ts
export async function publish(postId: number): Promise<void> {
  await db.table<Post>("posts")
    .where("id", "=", postId)
    .update({ status: "published", published_at: new Date().toISOString() });
}
```

## The public feed (published only, newest first, paginated)

`paginate()` returns the rows plus the metadata a UI needs (total pages, has-next)
in a single call.

```ts
export async function publicFeed(page = 1, perPage = 10) {
  return db.table<Post>("posts")
    .where("status", "=", "published")
    .orderBy("published_at", "DESC")
    .paginate(page, perPage);
  // → { items: Post[], total, totalPages, hasNext, hasPrev, ... }
}
```

## A single post by slug, with its author and comments

`with()` eager-loads relations in one batched query each — no N+1.

```ts
export async function getPostBySlug(slug: string) {
  const post = await db.table<Post>("posts")
    .where("slug", "=", slug)
    .where("status", "=", "published")
    .with("author", { table: "authors", foreignKey: "author_id", type: "one" })
    .with("comments", { table: "comments", foreignKey: "post_id", type: "many" })
    .first();

  if (!post) return null;

  // Count the view atomically (no read-modify-write race)
  await db.table<Post>("posts").where("id", "=", post.id).increment("views", 1);
  return post;
}
```

## Search posts

```ts
export async function searchPosts(term: string) {
  return db.table<Post>("posts")
    .where("status", "=", "published")
    .whereGroup((q) => {
      q.where("title", "LIKE", `%${term}%`)
        .orWhere("body", "LIKE", `%${term}%`);
    })
    .orderBy("published_at", "DESC")
    .limit(20)
    .get();
}
```

## Posting a comment inside a transaction

Comment + bump the post's comment activity together, all-or-nothing.

```ts
export async function addComment(postId: number, authorName: string, body: string) {
  await db.transaction(async (trx) => {
    const post = await trx.table<Post>("posts").find(postId);
    if (!post || post.status !== "published") {
      throw new Error("Cannot comment on a non-published post"); // → automatic ROLLBACK
    }
    await trx.table("comments").insert([{ post_id: postId, author_name: authorName, body }]);
  });
}
```

## Wiring it to an HTTP handler

The functions above are framework-agnostic. Here they are behind plain routes:

```ts
import express from "express";
const app = express();
app.use(express.json());

app.get("/posts", async (req, res) => {
  const page = Number(req.query.page ?? 1);
  res.json(await publicFeed(page));
});

app.get("/posts/:slug", async (req, res) => {
  const post = await getPostBySlug(req.params.slug);
  post ? res.json(post) : res.status(404).json({ error: "Not found" });
});

app.post("/posts/:id/comments", async (req, res) => {
  try {
    await addComment(Number(req.params.id), req.body.author, req.body.body);
    res.status(201).end();
  } catch (e) {
    res.status(400).json({ error: (e as Error).message });
  }
});

app.listen(3000);
```
