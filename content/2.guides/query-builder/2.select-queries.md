## Basic Selection

### select() - Specify Columns

The `select()` method allows you to specify which columns you want to retrieve from the table.

```typescript
// Select all columns (default)
const users = await db.table("users").get();

// Select all columns explicitly
const users = await db.table("users").select().get();

// Select specific columns
const users = await db.table("users").select(["id", "name", "email"]).get();

// Select a single column
const names = await db.table("users").select("name").get();
```

**Parameters:**

- `columns` (string[] | string, optional): Column names to select. Defaults to `*`

**Returns:** `Table` instance for method chaining

## Column Aliases

Use aliases to rename columns in the results:

```typescript
// Simple aliases
const users = await db
  .table("users")
  .select([
    "name as full_name",
    "email as email_address",
    "created_at as registered_on",
  ])
  .get();

// Mix columns with and without aliases
const posts = await db
  .table("posts")
  .select([
    "id",
    "title",
    "content as text",
    "user_id as author",
    "created_at as published_on",
  ])
  .get();
```

## SQL Expressions

You can use SQL expressions directly in your SELECT queries:

```typescript
// String expressions
const users = await db
  .table("users")
  .select([
    "name",
    "UPPER(email) as email_uppercase",
    'CONCAT(name, " - ", email) as name_email',
  ])
  .get();

// Date expressions
const posts = await db
  .table("posts")
  .select([
    "title",
    "YEAR(created_at) as publication_year",
    "MONTH(created_at) as publication_month",
    'DATE_FORMAT(created_at, "%d/%m/%Y") as formatted_date',
  ])
  .get();

// Mathematical expressions
const products = await db
  .table("products")
  .select([
    "name",
    "price",
    "price * 0.21 as tax",
    "price * 1.21 as price_with_tax",
    "ROUND(price * 1.21, 2) as final_price",
  ])
  .get();
```

## distinct() - Unique Values

Remove duplicates from results:

```typescript
// Get unique user states
const states = await db.table("users").select("state").distinct().get();

// Unique combinations of columns
const combinations = await db
  .table("orders")
  .select(["status", "priority"])
  .distinct()
  .get();

// With WHERE conditions
const activeCities = await db
  .table("users")
  .select("city")
  .distinct()
  .where("active", true)
  .get();
```

## Advanced SELECT Queries

### Subqueries in SELECT

```typescript
// Count posts per user
const usersWithCount = await db
  .table("users")
  .select([
    "users.*",
    db
      .table("posts")
      .count("*")
      .where("user_id", db.raw("users.id"))
      .as("total_posts"),
  ])
  .get();

// Last login date
const usersWithLastLogin = await db
  .table("users")
  .select([
    "id",
    "name",
    "email",
    db
      .table("logins")
      .select("login_date")
      .where("user_id", db.raw("users.id"))
      .orderBy("login_date", "desc")
      .limit(1)
      .as("last_login"),
  ])
  .get();
```

### Conditional Expressions

```typescript
// CASE WHEN in SELECT
const users = await db
  .table("users")
  .select([
    "name",
    "age",
    db.raw(`
      CASE 
        WHEN age < 18 THEN 'Minor'
        WHEN age BETWEEN 18 AND 65 THEN 'Adult'
        ELSE 'Senior'
      END as age_category
    `),
  ])
  .get();

// IF in MySQL
const products = await db
  .table("products")
  .select([
    "name",
    "stock",
    "price",
    db.raw('IF(stock > 0, "Available", "Out of Stock") as availability'),
  ])
  .get();
```

### Aggregation Functions in Subqueries

```typescript
// Compare with averages
const usersWithComparison = await db
  .table("users")
  .select([
    "name",
    "score",
    db.raw("(SELECT AVG(score) FROM users) as global_average"),
    db.raw(`
      CASE 
        WHEN score > (SELECT AVG(score) FROM users) 
        THEN 'Above average'
        ELSE 'Below average'
      END as comparison
    `),
  ])
  .get();
```

## Practical Examples

### 1. User Dashboard

```typescript
async function getUserDashboard(userId: number) {
  return await db
    .table("users")
    .select([
      "users.id",
      "users.name",
      "users.email",
      "users.created_at as registered_on",
      db.raw("DATEDIFF(NOW(), users.created_at) as days_registered"),

      // Count posts
      db.table("posts").count("*").where("user_id", userId).as("total_posts"),

      // Published posts
      db
        .table("posts")
        .count("*")
        .where("user_id", userId)
        .where("status", "published")
        .as("published_posts"),

      // Latest post
      db
        .table("posts")
        .select("title")
        .where("user_id", userId)
        .orderBy("created_at", "desc")
        .limit(1)
        .as("latest_post"),
    ])
    .where("users.id", userId)
    .first();
}
```

### 2. Sales Report

```typescript
async function getSalesReport(year: number) {
  return await db
    .table("orders")
    .select([
      db.raw("MONTH(order_date) as month"),
      db.raw("MONTHNAME(order_date) as month_name"),
      db.raw("COUNT(*) as total_orders"),
      db.raw("SUM(total) as total_sales"),
      db.raw("AVG(total) as average_sale"),
      db.raw("MIN(total) as minimum_sale"),
      db.raw("MAX(total) as maximum_sale"),
    ])
    .where(db.raw("YEAR(order_date)"), year)
    .where("status", "completed")
    .groupBy(db.raw("MONTH(order_date)"))
    .orderBy(db.raw("MONTH(order_date)"))
    .get();
}
```

### 3. Extended Product List

```typescript
async function getExtendedProducts() {
  return await db
    .table("products")
    .select([
      "products.id",
      "products.name",
      "products.description",
      "products.price",
      "products.stock",

      // Category information
      "categories.name as category",
      "categories.description as category_description",

      // Stock status
      db.raw(`
        CASE 
          WHEN stock = 0 THEN 'Out of Stock'
          WHEN stock < 10 THEN 'Low Stock'
          ELSE 'Available'
        END as stock_status
      `),

      // Discounted price
      db.raw("price * 0.9 as discounted_price"),

      // Total reviews
      db
        .table("reviews")
        .count("*")
        .where("product_id", db.raw("products.id"))
        .as("total_reviews"),

      // Average rating
      db
        .table("reviews")
        .avg("rating")
        .where("product_id", db.raw("products.id"))
        .as("average_rating"),
    ])
    .join("categories", "products.category_id", "=", "categories.id")
    .where("products.active", true)
    .orderBy("products.name")
    .get();
}
```

## SELECT Query Optimization

### 1. Select Only Necessary Columns

```typescript
// ❌ Avoid - selects all columns
const users = await db.table("users").get();

// ✅ Better - selects only necessary columns
const users = await db.table("users").select(["id", "name", "email"]).get();
```

### 2. Use Indexes in Expressions

```typescript
// ❌ Avoid - functions in WHERE prevent index usage
const users = await db
  .table("users")
  .where(db.raw("YEAR(created_at)"), 2023)
  .get();

// ✅ Better - use ranges to maintain index
const users = await db
  .table("users")
  .where("created_at", ">=", "2023-01-01")
  .where("created_at", "<", "2024-01-01")
  .get();
```

### 3. Limit Results

```typescript
// Always use LIMIT for test queries
const sample = await db
  .table("users")
  .select(["id", "name", "email"])
  .limit(10)
  .get();
```

## Types and Validations

```typescript
// Interface for typed results
interface User {
  id: number;
  name: string;
  email: string;
  created_at: Date;
}

// Function with types
async function getUsers(): Promise<User[]> {
  const users = await db
    .table("users")
    .select(["id", "name", "email", "created_at"])
    .get();

  return users as User[];
}
```
